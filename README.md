# АИС «Электронная библиотека»

Автоматизированная информационная система для ведения реестра книг и читателей.

## Ссылка на сайт

[Открыть приложение](https://asoloveva326.pythonanywhere.com)

## Технологии

- Python 3.11
- Flask 3.0
- SQLAlchemy
- SQLite
- Bootstrap 5
- EasyMDE (Markdown редактор)
- Bleach (санитайзер)

## Реализованная функциональность

Выполнено основное задание в полном объёме, а также индивидуальный вариант 4 — «Учёт истории посещений».

### Вариант 4 — особенности реализации

- **Популярные книги** — на главной странице отображается топ-5 наиболее просматриваемых книг за последние 3 месяца со ссылками на страницы книг
- **Недавно просмотренные** — на главной странице отображаются последние 5 просмотренных книг в обратном хронологическом порядке (работает для авторизованных и анонимных пользователей, скрыт если просмотров нет)
- **Счётчик просмотров** — обновляется автоматически при каждом открытии страницы книги, не более 10 просмотров в день от одного пользователя
- **Статистика** — доступна только администратору через пункт «Статистика» в навигации:
  - Журнал действий пользователей с пагинацией и экспортом в CSV
  - Статистика просмотров с фильтром по дате и экспортом в CSV
